  • Sitema gerenciador de Aumoxarifados Automatizado

    2021 - 2022


    Participei junto a outros 2 desenvolvedores, da criação de um sistema de gerenciamento automatizado do estoque de um dos Batalhoes do corpo de bombeiros do Paraná
    Esse sitema consistia em uma interface cliente-servidor desenvolvida utilizando as linguagens Django, Phyton e MySql. Para realizar o controle dos equipamentos do local foram utilizadas Tags NFC para facilitar o registro de entrada e saida dos mesmos. Desenvolvemos leitores de baixo custo utilizando microcontroladores da familia ESP
    Desenvolvemos tambem controle de acesso e debloqueio do local com autenticação e registro no servidor para facilitar a rastreabilidade
    Mais informações desse projeto podem ser encontradas neste Artigo, publicado pelo tenente do corpo de bombeiros Lucas Gross Link para o Artigo

  • Estação metereologica de baixo custo

    2017 - 2022

    Este projeto tinha como intuito a criação de uma estação metereologica visando o menor custo de produção possivel
    O projeto passou por diversas modicações durante seu desenvolvimento, dentre elas foi o modo de distibuição das informações. No pricipio as informações coletadas eram enviadas de forma autonoma ao perfil do twitter Ardulindo do tempo
    imagem do perfil do bot no twitter

    Apos realizamos uma mudança de plataforma de microcontroladores da familia Arduino para microprocessadores da familia ESP
    Nessa fase do desenvolvimento foi possivel levar o prototipo para alguns eventos cientificos, em que alguns até recebeu uma premiação Dentre os eventos apresentados estão:

    -XI Mostra de Soluções Para Uma Vida Melhor
    -1º Mostra Brasileira de Inovação, Pesquisa Científica e Empreendedorismo (MOBIPE)
    -III MOSTRA CIENTÍFICA ECOINOVAR (1º lugar na categoria Ensino Tecnico)
